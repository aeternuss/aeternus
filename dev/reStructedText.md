ReStructedText
==============

Link
----

``` {.sourceCode .rst}
`Title <http://link>`_
```

Hyperlink Targets
-----------------

``` {.sourceCode .rst}
External hyperlinks, like Python_.

.. _Python: http://www.python.org/
```

Table
-----

``` {.sourceCode .rst}
.. csv-table::
   :header: h1, h2, h3

   c1l1, c2l1, c3l1
   c1l2, c2l2, c3l3
```

Section Number
--------------

``` {.sourceCode .rst}
.. sectnum::
```

Define
------

``` {.sourceCode .rst}
define::

   documents
```

Footnotes
---------

``` {.sourceCode .rst}
Autonumbered footnotes are
possible, like using [#]_ and [#]_.

.. [#] This is the first one.
.. [#] This is the second one.

They may be assigned 'autonumber
labels' - for instance,
[#fourth]_ and [#third]_.

.. [#third] a.k.a. third_

.. [#fourth] a.k.a. fourth_
```
